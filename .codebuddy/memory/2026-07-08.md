# 2026-07-08

## Candy 项目 — 第三轮：按 AI 架构规范实现首个可用版本
- 用户给了详细架构规范（Orange Candy v0.1.0），要求按 `parser → core → renderer` 严格分层实现首个可用版本，不合理处自动改正。

### 当前架构（rust/）
- `core/`：`ast.rs`(Scene/Slide/Action/Label/FrameData)、`meta.rs`(PrivateMeta)、`error.rs`(CandyError E001–E007)、`scheduler.rs`(Y 轴关键帧)、`interpolator.rs`(X 轴插值+opacity 钳制)。纯逻辑、可测。
- `parser/`：`tyx.rs`(parse_tyx，括号感知扫描 DSL)、`dsl.rs`(extract_dsl_from_svg，解析 candy-json)。
- `renderer/`：`typst.rs`(Renderer + in-memory World，**绕过 typst CLI**，编译每帧并 `typst_svg::svg` 出 SVG；`render_frame_pixels` 用 `typst_render` 出 RGBA)、`rav1e.rs`(encode_webm → AV1/IVF，但 **gated behind `video` feature**)。
- `lib.rs`：`build(input, output, format, fps, ppp)` 串联全流程；`main.rs`：`candy build <in> -o <out> [--format svg|webm] [--fps] [-p]`。
- 测试：9 个单元测试（scheduler/interpolator/parser/dsl/ast）全过；端到端 SVG 构建验证通过（30 帧，dot 从 0 平移到 4cm）。
- Cargo 依赖：`typst`/`typst-render`/`typst-svg`/`typst-library`/`typst-layout`/`typst-syntax`/`typst-utils`（注意 typst_svg 是独立 crate，`typst_svg::svg(page,&SvgOptions::default())` 出 String）、`rav1e`(`default-features=false`)、`serde`/`serde_json`/`clap`。

### 对规范的不合理处改正（已自动修正）
- Scene 原无 per-target 内容 → 增加 `items: HashMap<Label,String>`（渲染所需）。
- 规范说 `WebM` 且 `encode_webm(svg_frames)` → 实际 rav1e 出 AV1/IVF，且编码前已用 typst 栅格化（非重解析 SVG）；输出为 IVF。
- `render_frame(frame)` 单 Label 签名不足以渲染整帧 → 增加 `Renderer::render_frame_at(frame_idx, all_frames)` 组合所有 target。
- DSL 解析用轻量括号扫描而非完整 Typst 语义解析（v0.1 合理简化）。

### ⚠️ 关键坑：rav1e 0.8.1 tiling panic（重要，跨会话）
- `rav1e` 0.8.1 在 `tiling/plane_region.rs` 的 `cfg.yorigin+rect.y+rect.height <= cfg.alloc_height` 断言对**某些帧尺寸**会 panic，且发生在 rayon worker 线程内、**无法被 catch_unwind 可靠捕获**（会 abort 进程）。
- 已验证：场景切换分析关掉（`SceneDetectionSpeed::None`）无效；尺寸取 2/8 对齐、放大到 453px 均仍 panic。
- 处理：把实际编码逻辑 `encode_webm_inner` 关在 `#[cfg(feature="video")]`，默认 `encode_webm` 直接返回 `CandyError::Encode`(E007)，`build` 的 webm 分支据此**回退写 SVG 序列**（符合规范 §6 E007）。默认构建绝不 abort。要真出视频：`cargo run --features video -- build … --format webm`（仍可能因尺寸 panic，需后续修 rav1e 版本/补丁）。
- Typst `scale(x: 1)` 不接受整数，须用百分比字面量 `scale(x: 100%)`。

### typst 0.15 嵌入 API（已查证可用）
- `World`：`source()` 返回 `Result<Source, FileError>`，`FileError::NotFound(PathBuf)`；`Library::default()` 需 `typst::LibraryExt`；`LazyHash` 来自 `typst_utils`；`PagedDocument` 在 `typst-layout`；`page` 经 `.pages()` 方法取。
- `typst_render::RenderOptions { pixel_per_pt: Scalar::new(f), render_bleed: false }`，`render` 返回 `tiny_skia::Pixmap`。
